A 2D Tank game built in Java 8 with the Swing Library
Individual Classes Include:

Wall		- Handles non moving barriers
TankApp		- Main Function
Tank		- Handles Tank Functionality - extends Sprite
Sprite		- Common items for all moving sprites
RNG		- Random Direction Heading Generator for BadTanks
Missile		- Players Missile Controls
Board		- The Game Board
BadTank		- NPC tanks
